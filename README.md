# NGINX Stream Router Installer

This is a simple automated installer for NGINX stream (TCP/SNI) routing. It sets up everything required to quickly deploy a reverse proxy without manual configuration.

## Features:

- Automatic nginx installation
- Stream module setup
- SNI-based routing
- IPv4 + IPv6 support
- Logging enabled
- Interactive configuration

## Installation:

```bash
curl -O curl -O https://raw.githubusercontent.com/conjureseizures/nginx-stream-router-installer/main/install.sh
chmod +x install.sh
./install.sh
chmod +x install.sh
./install.sh
```

## Requirements:

Ubuntu/Debian
Root or sudo access
