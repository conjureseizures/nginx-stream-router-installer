# NGINX Stream Router Installer

Automated NGINX stream/SNI routing installer for fast TCP proxy deployment with configurable domains, ports, and upstream targets.

## Features:

- Automatic nginx installation
- Stream module setup
- SNI-based routing
- IPv4 + IPv6 support
- Logging enabled
- Interactive configuration

## Installation:

```bash
curl -O curl -O https://raw.githubusercontent.com/conjureseizures/nginx-stream-router-installer/main/install.sh
chmod +x install.sh
./install.sh
chmod +x install.sh
./install.sh
```

## Requirements:

Ubuntu/Debian
Root or sudo access
